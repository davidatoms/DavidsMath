## About This Project

This repository contains mathematical research and formal proofs exploring:
- Geometric-Quantum scaling connections
- Derivative analysis of scaling algorithms
- Connections between geometry, quantum mechanics, and information theory
- Formal proofs in Lean 4
- Computational implementations in Julia
